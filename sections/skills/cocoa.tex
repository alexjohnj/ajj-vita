\textbf{iOS Development}
\begin{itemize}[leftmargin=0mm]
\item Working in \skill{Swift} and \skill{Objective-C} both independently and in
  mixed codebases.
\item Applying \skill{object-orientated}, \skill{protocol orientated} and
  \skill{functional programming}\\paradigms where appropriate to create
  maintainable and reusable code.
\item Designing internal application architecture using \skill{MVC} and variants
  like \skill{MVC-C} and \skill{MVC-N}. I also haves some exposure to \skill{MVVM}.
\item Networking and interacting with \skill{RESTful} web services using
  NSURLSession and Swift's Codable protocols.
\item Extensive knowledge of \skill{Core Data}, its \skill{concurrency} model
  and efficiently importing data from web services.
\item Profiling Core Data using \skill{Instruments} and designing models for
  fast, Unicode aware search predicates.
\item \skill{Unit testing} using \skill{XCTest} including testing asynchronous
  networking and persistence methods.
\item Managing project dependencies using \skill{Carthage} and \skill{CocoaPods}
  as well as distributing open-source frameworks using \skill{Carthage}.
\end{itemize}
